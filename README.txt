# GD-Mods
Hola, eh decidido crear esta seccion para meter mods estilo .dll aqui

Tutorial:
1. Primero tienes que hacer el tutorial del modloader, si no lo haces no funcionara nada de esto
2. Creas una carpeta llamada "mods"
3. Descargas algun .dll que este en mi cuenta de github
4. Mete ese .dll en la carpeta de mods
5. Abre el juego
